# StudyTable Project

## Overview
This is a responsive **AI-powered educational platform** designed to help students prepare for **JEE, NEET, and Board exams**. The application provides a seamless user experience with modern animations using **Framer Motion**, and is built with **React.js** and **Tailwind CSS** for a sleek UI.
